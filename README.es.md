# Acerca de

Este repositorio contiene el código fuente de mi portfolio personal.

El objetivo de este proyecto no es solo presentar mi recorrido profesional, sino también mostrar mi forma de pensar, diseñar y desarrollar soluciones digitales.

El sitio está disponible en **español** e **inglés**, cuenta con diseño responsive, modo claro/oscuro y acceso directo a mis proyectos.
